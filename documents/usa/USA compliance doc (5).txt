5. RESO Web API Technical Standards
The technical architecture of the U.S. real estate market is now built exclusively on the RESO Web API standard. The legacy Real Estate Transaction Standard (RETS) is fully deprecated and must not be used in any 2026 integration.
5.1 Data Dictionary 2.0 Certification
NAR-affiliated MLSs were required to achieve RESO Data Dictionary 2.0 certification by April 2025. Significant updates include standardized status values (Active, Pending, Closed), detailed environmental hazard fields (fire/flood risk), and transition to 'Primary' terminology for property features.
UPCOMING: RESO is moving toward Data Dictionary 2.1 and 2.2 releases, which introduce fields for digital staging and AI-enhanced photo disclosure including MediaDigitalEnhancementYN (see Section 11).
